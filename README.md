# Pixi Starter Kit

This repo contains everything needed to start building a [Pixi JS](https://pixijs.com) app with:

- Parcel for bundling and local dev server
- TypeScript
- ESLint
- Prettier
- Jest, with canvas / WebGL mocks
- Normalize CSS reset

## Getting Started

```
yarn install
yarn start
```
